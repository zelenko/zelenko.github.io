Prefered technologies for Web, ERP, and Customer solutions

Here are some technologies, tools, and best practices I observed over the years, and absorbed for implementing Web, ERP, and Customer solutions.

Text Editors

Years ago I used Macromedia Dreamweaver, Notepad++, Visual Studio Code. Now I prefer GoLand!

GoLand is a new commercial IDE by JetBrains aimed at providing an ergonomic environment for Go development.

Operating Systems

Now I use Windows 10, CentOS, Red Hat, Debian, Ubuntu, and Amazon Machine Images (AMI).

Years ago I started with Windows 95, then migrated to Windows 98, ME, and NT.

E-Commerce & CMS

Magento, WooCommerce, OpenCart, Wordpress, QOR

I also used these: Gistlog, Drupal, eZ Publish, osCommerce, and gohugo.io

Payment Processing

Paypal, Authorize.net, Amazon Pay, Google Wallet

Avalara for tax processing

Databases

MongoDB, Rocket U2 UniVerse, PostgreSQL, MariaDB

NoSQL is preferred, especially MongoDB.

Help Desk System

Dell KACE K1000 Systems, ZenDesk, ManageEngine ServiceDesk Plus, Jira.

You have to support your systems, so these are the tools.

ERP Systems

Eclipse ERP, SAP, Microsoft Dynamics, Prophet 21, Openbravo, Covestone

Eclipse ERP was designed for industrial distribution companies in HVAC, PVF, Electrical, and related industries.

Domain Registration

Godaddy, Network Solutions

Get a domain that most people can type without asking how it's spelled.

Cloud computing

Amazon Elastic Compute Cloud (Amazon EC2), Google App Engine, Rackspace, Digital Ocean

TurnkeyLinux appliances allow for fast installations.

Web Server Software

HttpRouter from Julien Schmidt, because it is lightweight offers high performance.

In the past I used Nginx, Lighttpd, Apache, ISS.

Development

Go, PHP, JavaScript. I love the Go programming language

Hybrid apps for mobile devices using HTML5 (aka Progressive Web Apps).
